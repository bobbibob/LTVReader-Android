## Где LTV Reader выигрывает

1. **Бесплатность**: open-source, без подписки, без рекламы.
2. **Приватность**: on-device Kokoro, ваши тексты не уходят в облако.
3. **Гибкость**: LTV-разметка даёт точный контроль над озвучкой.
4. **Подкасты**: встроенный микшер с ducking.
5. **Кастомность**: Custom HTTP движок + remote host для своих моделей.
6. **Размер**: самый маленький APK в категории.

## Где проигрывает

1. **Качество голосов**: Kokoro уступает ElevenLabs и профессиональным
   актёрам. Но для длинных книг разница меньше.
2. **Нет Whisper-верификации** (на устройстве).
3. **Нет iOS-версии**.
4. **Нет iCloud-синхронизации** проектов.
